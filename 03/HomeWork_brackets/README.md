"Скобочные последовательности" (на рекурсию)
============================================
На входе: N - количество пар скобок

Нужно:
Вывести все правильные скобочные
последовательности из обычных скобок '(' ')'
Другие виды скобок: {} []
из N пар скобок
Например:
N = 1
Ответ:
```
()
```
N = 2
```
(())
()()
```
