OpenSource (открытый код)
=========================
* SQLite - один модуль С/C++
  - встроенная в Firefox, Chrome
    Android
  - Драйвер БД входит в Qt 
  - БД хранится в файле
  - Никакого серверного ПО 
   не нужно
  - CREATE / DROP
  - SELECT / INSERT / UPDATE / DELETE
  - Не работает RIGHT JOIN, 
    выражается через LEFT JOIN
  - Нет хранимых процедур
  - связи между таблицами на 
    уровне логики приложения  
* MySQL (MariaDB)
  - Компактная (размер ~50 Мб)
  - Полноценная серверная
  - Есть поддержка хранимых 
    процедур только на SQL
  - Каскадная обработка данных
  - Autoincrement
  - для 32-битных версий 
   ограничения по размерам таблиц, индексов
   порядка 4 Гб
* PostgreSQL (Postgres)
  - размер порядка 200 Мб
  - поддержка хранимых процедур
    SQL, Python, Ruby, можно 
    подключать модули на C++...
  - развитая система прав/
    пользователей/групп
  - последовательности 
    (sequences)
  - Ограничение на таблицу порядка 
    16 Тб
* MongoDB - noSQL / документ-
  ориентированные
